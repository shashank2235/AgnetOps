# AgentOps Runtime v0.1.1

UI refresh release for the AgentOps Runtime dashboard.

## Highlights

- Redesigned operational dashboard with stronger visual hierarchy
- Hero section, KPI cards, and clearer execution feed
- More polished approval queue and status presentation
- Improved mobile layout and card styling
- Updated README preview asset to reflect the new interface

## Scope

This release focuses on frontend presentation and operator experience. Backend APIs and runtime architecture remain unchanged from v0.1.0.
