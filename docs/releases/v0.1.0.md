# AgentOps Runtime v0.1.0

Initial public release of AgentOps Runtime.

## Highlights

- LangGraph-based multi-step agent runtime
- Step checkpointing and resume support
- Recovery workflows for retry, rollback, and replay
- Guardrails policy engine with human-approval routing
- OpenTelemetry instrumentation with Jaeger-compatible OTLP export
- Evaluation service with groundedness, relevance, and context precision scoring
- Hybrid memory pattern for vector and graph context
- React dashboard for run visibility and approval operations

## Included APIs

- Agent execution and lifecycle management
- Checkpoint query and resume
- Approval task queue and decisions
- Guardrail check endpoint
- Evaluation and metrics endpoints

## Deployment and Ops

- Docker Compose for full local stack
- Optional Kubernetes manifests
- CI workflow for backend tests and frontend build

## Known Limitations

- External service adapters are partially mocked in baseline release
- Replay worker metadata is implemented; full deterministic re-execution is iterative

## Upgrade Notes

- This is the first release baseline. No migration required.
